# Local Real Mode Activation Path

本地真实模式是 OpenButler 的核心能力，但它不能在公开 Vercel 页面里被暗示为已经连接。

## 用户需要知道什么

真实本地模式需要三步：

1. 在自己的电脑上运行 OpenButler 本地版。
2. 主动授权要读取的本地线索，例如电脑活动。
3. 先预览会读取什么，再决定是否生成今日概览、时间线和提醒。

## 会读取什么

只读取用户明确授权的本地线索。当前主要是本机电脑活动和 OpenButler 派生数据。

## 不会读取什么

- 不会自动读取真实相册。
- 不会自动复制截图。
- 不会上传本地数据。
- 不会读取密码、Cookie、Token、验证码。
- 不会替用户确认远程仓库、部署、接口或任务系统状态。

## 普通页面文案

使用：

- `了解本地模式`
- `真实模式需要你在本机运行，并主动授权。`
- `授权后会得到今日概览、时间线、管家提醒和可复核依据。`

避免：

- `连接本地数据源`
- `导入 PC Activity`
- `MineContext`
- `godview`

这些词可以留在高级与实验室中。
