# Demo / Local Mode Boundary

OpenButler 现在有两种使用语境，必须在产品里说清楚。

## 公开样例体验

Vercel 线上版是公开 Demo。它只展示产品如何整理线索、提醒重点和解释依据。

普通页面统一使用：

- `样例体验，未读取你的真实数据。`
- `线上版本不会读取你的真实本机活动。`
- `想使用真实数据，需要在自己的电脑上启动本地版。`

不要在普通路径使用：

- `Vercel demo`
- `mock`
- `seed`
- `fixture`
- `导入`
- `运行验证`
- `连接本地数据源`

这些词会让用户误以为网页正在访问本机，或者误以为产品只是工程演示。

## 本轮隐私边界

本轮不读取真实 MineContext 数据。
本轮不复制截图。
本轮不上传数据。
本轮不调用外部模型。
本轮不写入外部系统。

## CTA 规则

公开 Demo 的普通 CTA 使用用户任务：

- `看今天建议`
- `查看时间线`
- `问管家`
- `查看依据`
- `了解本地模式`

真实本地模式的入口先解释边界，再引导用户进入本机设置。不要把公开网页里的普通用户直接推到技术配置页。
