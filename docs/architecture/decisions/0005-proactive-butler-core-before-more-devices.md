# 0005 Proactive Butler Core Before More Devices

## Status

Accepted.

## Context

Adding more sensors is tempting, but the product risk is not sensing volume; it is whether OpenButler can turn authorized evidence into useful daily guidance.

## Decision

Stabilize Proactive Butler Core before adding more hardware or mobile complexity.

## Consequences

- No new acoustic imager, thermal camera, or USB peripheral integration in the current stage.
- Invest in timeline, metrics, insights, inbox, feedback, docs, tests, and privacy boundaries.
- Future devices must plug into the same event and evidence pipeline.
